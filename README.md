# amazon-clone-homepage
An Amazon-inspired frontend clone built using HTML and CSS.
This project was created for frontend practice and learning purposes.

## Features
* Responsive navigation bar
* Amazon-style hero section
* Product category boxes
* Footer section similar to Amazon
* Font Awesome icons integration
* Clean and modern UI design

## Technologies Used
* HTML5
* CSS3
* Font Awesome

## Project Preview
The project includes:
* Navigation Bar
* Search Bar
* Product Sections
* Hero Banner
* Footer Panel
